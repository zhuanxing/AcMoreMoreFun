## P1.Acfun_更多的直播跳转.js

不用再担心因为D得太多而每次都要到直播站才能点开你要单推的主播了

安装之后可以直接看到所有关注且正在直播的UP了！

**注：因为涉及跨域资源，请在油猴询问【一个用户脚本试图访问跨源资源。】时候选择【总是允许/总是允许该域名】，否则无法正常获得信息。**
